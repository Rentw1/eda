# 🥗 Calorie Telegram Bot

Бот считает калории по фото и текстовому описанию. Использует Gemini 2.0 Flash Vision (бесплатно).

## Что умеет

- 📸 Отправить фото еды → бот определяет блюдо и КБЖУ
- ✏️ Написать «200г гречки» → бот посчитает
- Выбрать граммовку кнопками или ввести своё
- 📊 Статистика за день и неделю с прогресс-баром
- 🎯 Установить цель по калориям
- ↩️ Отменить последнюю запись

---

## Шаг 1 — Получить токены

### Telegram Bot Token
1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. `/newbot` → придумай имя → получи токен вида `7123456789:AAF...`

### Gemini API Key (бесплатно)
1. Зайди на [aistudio.google.com](https://aistudio.google.com)
2. Нажми **Get API key** → **Create API key**
3. Скопируй ключ вида `AIza...`
4. Бесплатный лимит: **1500 запросов/день** — для одного человека хватит

---

## Шаг 2 — Деплой на Render (бесплатно)

1. Залей папку на GitHub (новый репо)
2. Зайди на [render.com](https://render.com) → **New → Blueprint**
3. Подключи свой репо → Render увидит `render.yaml`
4. Добавь переменные окружения:
   - `BOT_TOKEN` = твой токен от BotFather
   - `GEMINI_API_KEY` = ключ от Google AI Studio
5. Нажми **Apply** → бот запустится

> **Важно:** выбирай тип `Worker` (не Web Service) — тогда бот не засыпает.
> На бесплатном плане worker работает без ограничений по времени.

## Локальный запуск (для теста)

```bash
pip install -r requirements.txt
export BOT_TOKEN="твой_токен"
export GEMINI_API_KEY="твой_ключ"
python bot.py
```

База данных `calories.db` создаётся автоматически в папке проекта.

---

## Структура проекта

```
bot.py          — точка входа, запуск polling
config.py       — чтение env переменных
database.py     — SQLite: приёмы пищи, цели
gemini.py       — запросы к Gemini Vision API
handlers.py     — вся логика бота (хендлеры)
keyboards.py    — инлайн и reply клавиатуры
states.py       — FSM состояния
render.yaml     — конфиг для Render.com
```

## Точность

Gemini 2.0 Flash даёт погрешность ~±30–40% на оценке веса порции по фото.  
Это ±100–150 ккал — нормально для личного трекера.  
Если хочешь точнее — всегда можно вручную скорректировать граммы кнопками.
